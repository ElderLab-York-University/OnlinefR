The MCMLSD algorithm is taken from https://www.elderlab.yorku.ca/mcmlsd/
Almazen, E.J., Tal, R., Qian, Y. & Elder, J.H. (2017) A dynamic programming approach to line segment detection.  IEEE Conference on Computer Vision and Pattern Recognition (CVPR), 2031-2039.

The LSD algorithm is taken from 
Von Gioi, Rafael Grompone, et al. "LSD: A fast line segment detector with a false detection control." IEEE transactions on pattern analysis and machine intelligence 32.4 (2008): 722-732.

Please run demo_Manhattan.m for the MCMLSD version
Please run demo_Manhattan_fast.m for the LSD version.
